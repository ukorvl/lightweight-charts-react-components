---
title: Series Primitives
description: Attach Lightweight Charts series plugins through a React component boundary.
---

Use `SeriesPrimitive` when you need custom drawing that belongs to exactly one series. The
component accepts either an already-created upstream plugin instance or a `render`
function that receives the current chart and series APIs.
