---
title: Watermarks
description: Add text or image watermarks to the chart using declarative components.
---

The watermark components stay close to the upstream plugin model while keeping the React
composition story simple. Add a text watermark with `WatermarkText` or supply an image
source with `WatermarkImage`.

<CodeExample path="watermark.tsx" title="Text watermark" />

Use watermarks for branding, scenario labels, or background status indicators without
having to drop into imperative chart plugin code.

See the full
[Watermark sample](https://github.com/ukorvl/lightweight-charts-react-components/tree/main/examples/src/samples/Watermark)
for more styling ideas.
